# Pong
This classic mini-game is included as part of the abandoned Kano OS.

# Installation
```
sudo apt update
sudo apt upgrade
sudo apt install love zip
cd ~
git clone https://github.com/bggardner/kano-pong.git
cd kano-pong
zip -9 -r pong.love .
```

# Usage
`love ~/kano-pong/pong.love`
